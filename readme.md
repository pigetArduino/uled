# Wiring
Warning : Do not connect more than 5 leds.   
* D6 --> RESISTOR (470Ohm) DI
* +5V --> 5V
* GND --> GND

You can add a 1000uf capacitor on +5V especially if you want to manage more leds   
Use a dedicated power supply for this.
